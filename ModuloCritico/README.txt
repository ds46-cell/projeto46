Módulo: Risk Engine / Audit Chain

Objetivo:
Este módulo demonstra como o sistema reduz riscos e evita perdas financeiras para a instituição bancária.

Como rodar:
1. Certifique-se de ter Node.js instalado (versão >= 18).
2. Entre na pasta src/:
   cd src/
3. Instale as dependências:
   npm install
4. Inicie o módulo:
   node index.js
5. Os resultados, logs e alertas serão exibidos no console ou painel conforme configuração.
6. Para dados de teste, utilize a pasta TestData/ (vazia ou com arquivos simulados).

Objetivo do piloto:
Executar o módulo por 7 dias para demonstrar impacto estratégico e retorno financeiro.
A intermediária poderá gerar dados simulados para testar o módulo.

Observações:
- Este módulo é apenas parte do sistema completo.
- O sistema total inclui auditoria, monitoramento e análise estratégica de decisões.
